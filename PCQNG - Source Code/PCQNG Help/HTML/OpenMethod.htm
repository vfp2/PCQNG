Open()

Return Value:
VOID

C++ Declaration:
HRESULT Open(void)

Remarks:
Open() is a COM method. Open() creates a PCQNG device on the system and must be called once before any other PCQNG method or property is used. Calling other methods or properties before the PCQNG device has been opened will result in a PCQNG_E_DEVICE_NOT_OPEN exception. Only one PCQNG device may be opened on a single machine at a time. Calling Open() on a machine with the PCQNG already in use (opened) will result in a PCQNG_E_DEVICE_UNAVAILABLE exception. Call the Close() method when the PCQNG resource is no longer needed.
