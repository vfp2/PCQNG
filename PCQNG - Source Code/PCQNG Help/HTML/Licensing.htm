Licensing


Licensing Information
All licensing is subject to the End-User License Agreement. The PCQNG 2.0 software allows for a 14 day evaluation period. After the 14 day evaluation period the PCQNG 2.0 stops functioning. You must purchase a license to continue using the PCQNG 2.0 after the evaluation period. Licenses can be purchased online at http://www.ComScire.com.

Purchased licenses will be sent via email directly after purchase. If you have not received a license in your email within 24 hours after purchase, please contact ComScire Support.

The email from ComScire will contain a "User ID" and a "Product Key" needed to activate the license. The "User ID" is usually the email address the license was sent to.

Activating the License
(1) Log in as an Administrator, if you have not already done so.
(2) Install the PCQNG 2.0 software, if you have not already done so.
(3) Close all applications running the PCQNG 2.0 true random number generator.
(4) Double-click to run "PCQNG 2.0 Licensing" located on your Desktop.
(5) Click "No" to display License Activation Dialog.
(6) Enter "User ID" and "Product Key" as prompted.

Note: If "PCQNG 2.0 Licensing" does not display any windows or errors when run, the license has probably been previously activated.
