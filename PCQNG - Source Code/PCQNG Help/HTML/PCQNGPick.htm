PCQNGPick 1.0 Application

The PCQNGPick application simulates a random lottery type drawing using the PCQNG 2.0.

Change the numbers to the desired settings, then click on "CREATE PICK" to generate the output. The output can be copied and pasted to another application by highlighting the text and using the key combo [Ctrl + C] to copy and [Ctrl + V] to paste.
