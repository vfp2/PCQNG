RandNormal

Return Value:
The return value is a DOUBLE (VT_R8) random normal number with mean zero and standard deviation = 1.0

C++ Declaration:
HRESULT RandUniform([out, retval] DOUBLE* pVal)

Remarks:
RandNormal is a COM property. RandNormal numbers are produced by passing uniform numbers, with 48 PCQNG generated bits each, through the Box-Mueller method.
