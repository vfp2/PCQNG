Frequently Asked Questions


Q. What is the PCQNG 2.0?
A. The PCQNG is a true random number generator using hardware already present in your computer to produce high quality true entropy bits.

Q. What are the minimum system requirements for the PCQNG 2.0?
A. The minimum system is a 300 MHz Pentium class computer running Windows 98 or Windows 2000 (or later).

Q. How do I use the PCQNG 2.0?
A. The PCQNG can be programmatically accessed via an ActiveX control. Please refer to the programmers guide for more details.

Q. How good are the random numbers produced by the PCQNG 2.0?
A. The PCQNG's output has been tested extensively by the most stringent tests available and it has not failed any test for randomness in tests utilizing up to 1010 bits.

Q. Where can I find a faster true random number generator?
A. ComScire has an external hardware TRNG that produces 1 million high quality bits per second. For more information visit www.ComScire.com.
