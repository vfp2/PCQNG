Return Values

The following lists the values and text descriptions that can be returned by the PCQNG ActiveX control. Error values are negative and can not be combined.

PCQNG_S_OK 00044500h
    PCQNG reports success.
 
PCQNG_E_GENERAL_FAILURE 80044501h
    PCQNG general error.
 
PCQNG_E_IO_ARRAY_OVERSIZED      80044504h
    PCQNG read array size exceeds 8192 bytes.
 
PCQNG_E_DEVICE_UNAVAILABLE 8004450Ah
    PCQNG unavailable or already in use.
 
PCQNG_E_DEVICE_NOT_OPEN 8004450Bh
    PCQNG device has not been opened yet.
 
PCQNG_E_INVALID_SYSTEM 8004450Dh
    PCQNG incompatible with system.
 
PCQNG_E_INVALID_MODE 80044511h
    PCQNG incompatible mode.
 
S_OK 00000000h
    No error occurred.
