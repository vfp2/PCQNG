SetMode(Mode)

Parameter:
Mode: Bit rate multiplier or special mode (see below) as a FLOAT (VT_R4).

Return Value:
VOID

C++ Declaration:
HRESULT SetMode([in] FLOAT Mode)

Remarks:
SetMode() is a COM method. SetMode() is normally used to set the bit rate multiplier for the PCQNG device. The base bit rate is 1000 bps and the default mode is 1. To get a bit rate of 2000 bps, set the mode to 2 (i.e. 2 x 1000 bps). To get a bit rate of 500 bps, set the mode to 0.5 (i.e. 0.5 x 1000 bps). The following is a table of valid normal modes:

Special modes that may be set with SetMode():

DMMI mode = -1
Minimal processing of raw bits for DMMI processing at a bit rate of 1000 bps. Statistical randomness is not guaranteed in this mode.

Raw bit mode = 0
Raw bits - no processing. Before entropy extraction and statistical correction. The raw bit rate is 8000 bps. Statistical randomness is not expected in this mode.
