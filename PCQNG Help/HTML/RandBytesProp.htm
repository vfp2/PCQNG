RandBytes(Length)

Parameter:
Length: Number of bytes to be returned from RandBytes. Must not exceed 8192.

Return Value:
RandBytes returns a VARIANT SAFEARRAY of BYTES (VT_ARRAY of VT_UI1).

C++ Declaration:
HRESULT RandBytes([in] LONG Length, [out, retval] VARIANT* pVal)

Remarks:
RandBytes is a COM property. Each byte contains 8 PCQNG generated bits. If  Length exceeds 8192 requested bytes, the control will return the PCQNG_E_IO_ARRAY_OVERSIZED exception code. For information on using Variant SAFEARRAY's, please consult documentation specific to your programming environment.
