Basic Programming Examples

Basic programming examples are provided in several different languages to get you started quickly. The examples show how to use the PCQNG ActiveX interface to get random 32-bit integers, uniform numbers [0,1), and normally distributed numbers.

The PCQNG 2.0 is implemented as an ActiveX control. As such, it is widely supported among different environments and languages. If you do not find a basic example suitable for your particular environment, consult your developers guide on how to use ActiveX controls.
