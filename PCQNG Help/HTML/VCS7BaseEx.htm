Visual Basic 7.0 and Visual C# 7.0 Basic Example

This example shows how to use the PCQNG ActiveX interface to get random 32-bit integers, uniform numbers [0,1), and normally distributed numbers using the ComScire PCQNG True Random Number Generator. Visual Basic 7.0 and Visual C# 7.0 examples are almost identical, except for Step 5 where each gets its respective code.

Click on for a screen shot of each step.
Step 1 Create a new C# Windows Application
Start a new Visual C# project. Select "Windows Application" as the template. Name the project and click OK.

Step 2 Insert the PCQNG ActiveX control
In the Toolbox click the "Components" tab to open it. Then, right click in the Toolbox and open "Customize Toolbox...".
In the "COM Components" find and place a checkmark next to "ComScire PCQNG Control 2.0". Click OK to add a PCQNG control item to the Toolbox.
Click on the PCQNG icon in the Toolbox, then click on the form to add the PCQNG control to your form.
VB only: Right-click on the PCQNG icon on your form, select "Properties". Confirm that the "(Name)" property is "AxPCQNG1". If not, change it.
C# only: Right-click on the PCQNG icon on your form, select "Properties". Confirm that the "(Name)" property is "axPCQNG1". If not, change it.

Step 3 Add radio buttons for data type selection
Click on the "Windows Forms" tab in the Toolbox.
Add a group box to the form; then, place 3 radio buttons into the group box.
Right click on the group box and select "Properties". In the Properties area, Change the group box "(Name)" and "Text" properties to "Selection".
Select each radio button to change their properties. For each radio button change the "(Name)" and "Text" properties to "Uniform", "Normal", "Int32" respectively.

Step 4 Add labels for output
Add two Labels to the dialog.
Right click on the first label Label and select "Properties". Change the "Text" property of the first label to "Random Value:".
Click on the second Label and change its "(Name)" and "Text" properties to "Out".

VB Step 5 Add code to a "Go" button (see below for C# Step 5)
Add a Button to the dialog and change the Button "(Name)" and "Text" properties to "Go".
Double click on the "Go" button to edit the code section. Edit the Go_Click section as follows:

Private Sub Go_Click(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles Go.Click

    AxPCQNG1.Open()

    If Int32.Checked Then
        Out.Text = AxPCQNG1.RandInt32.ToString
    End If

    If Uniform.Checked Then
        Out.Text = AxPCQNG1.RandUniform.ToString
    End If

    If Normal.Checked Then
        Out.Text = AxPCQNG1.RandNormal.ToString
    End If

    AxPCQNG1.Close()

End Sub

C# Step 5 Add code to a "Go" button (see above for VB Step 5)
Add a Button to the dialog and change the Button "(Name)" and "Text" properties to "Go".
Double click on the "Go" button to edit the code section. Edit the Go_Click section as follows:

private void Go_Click(object sender, System.EventArgs e)
{
      axPCQNG1.Open();

      if (Int32.Checked)
            Out.Text = axPCQNG1.RandInt32.ToString();

      if (Uniform.Checked)
            Out.Text = axPCQNG1.RandUniform.ToString();

      if (Normal.Checked)
            Out.Text = axPCQNG1.RandNormal.ToString();

     axPCQNG1.Close();
}

Step 6 Compile and run
Click on the Debug menu and select "Start Without Debugging" to run the application. If the compiler finds errors, check to make sure the code appears exactly as shown in step 5.
Select the type of random number you wish to generate and press "Go" to generate a new random number.
